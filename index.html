My Projects

Irrupt | A Loot-based ARPG

Irrupt is a mobile game, developed in Godot 3.X Created for Android and iOS, created over a 6~ month period Lots of systems, combat, quests, shop, items, weapon and enemy progression, automatic dungeon generation, and an open world.

AltruVocab

Altru Vocab is a learning app I solo developed in Unity, and published to Android and iOS. The game was created for a client following their specifications.

Soulslike Prototype | made in 1 week

This is a Soulslike prototype created for a client in 1 week--which is my excuse for the ugly animations :) It features a character state machine, basic enemy AI, joystick + keyboard support. Made with assets from the Unity Asset store

2D MMORPG

This is my dream game. A re-make of Argentum Online, a 2D MMORPG popular in Argentina, created in 1999. It is developed in Godot, using the built-in ENet networking features that come with the engine. My goal in life is to become a millionaire so I can work on this project full time :D

Hecarim Run

Hecarim Run is an endless runner, which draws inspiration heavily from Robot Unicorn Attack, featuring Hecarim, a champion from League of Legends. At the time, Riot allowed users to create anything that used their IP, as long as there was no direct monetization or paywalls. Made in Unity and published on Android.

Wobbly Brick Breaker

This was my first published game. After following a tutorial and learning Unity3D, I thought with some changes the game would have potential. The end result was very fun to play, and it gained +10k downloads on the Google Play Store.
